What is this?

It's a website I'm making to learn a little bit more about API's, manipulating data, graphing, etc.



What is it built with?

It uses Node and Express for the backend, and Bootstrap for the front end. I use C3, D3, Datamaps and Topojson for the graphs. I use Moment to make dealing with dates easier.
